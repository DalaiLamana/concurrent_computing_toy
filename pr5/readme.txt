En la parte uno:
   El cliente toma tres argumentos: la ip del server, el puerto y
   el nombre del archivo que se quiere recoger.
   El servidor toma un argumento: el puerto.
   Para compilar hay que estar en la carpeta que contenga a uno de los archivos
   y ejecutar:
   javac Servidor.java o javac Cliente.java


En la parte dos:
   El servidor toma dos argumentos: La direccion del server y el puerto.
   El cliente toma tres argumentos: La direccion local,
   la direccion del server y el puerto

Ej: java src.Cliente localhost localhost 5550
Hay que poner un puerto suficientemente grande.
Para compilar hay que estar en la carpeta que contenga
la carpeta src y mensajes y ejecutar:
javac src/Cliente.java o javac src/Servidor.java
